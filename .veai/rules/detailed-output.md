# detailed-output

Provide detailed explanations when reporting work and decisions.

- Include the relevant context, trade-offs, and reasoning behind non-obvious choices.
- Summarize changed files and the behavior each change enables.
- Report verification commands and quote their important output.
- Keep details structured with headings or bullets so the user can skim or drill down.
